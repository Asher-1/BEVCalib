## 2. 核心模块详解

### 2.1 主模型：`bev_calib.py`

#### BEVCalib 类

整个模型的主类，整合图像分支、点云分支、特征融合和标定预测。

```python
class BEVCalib(nn.Module):
    def __init__(self,
                 num_heads=8,
                 num_layers=2,
                 deformable=True,
                 bev_encoder=False,
                 img_shape=None,
                 rotation_only=False,
                 enable_axis_loss=False,
                 weight_axis_rotation=0.3,
                 axis_weights=(3.0, 1.5, 1.0),
                 tail_topk_ratio=0.1,
                 tail_weight=0.5,
                 drop_path_rate=0.1,
                 head_dropout=0.1):
```

**主要组件**：

| 组件 | 类型 | 作用 |
|------|------|------|
| `img_branch` | Cam2BEV | 图像 → BEV 特征 |
| `pc_branch` | Lidar2BEV | 点云 → BEV 特征 |
| `conv_fuser` | ConvFuser | 拼接 + 1×1 卷积融合 |
| `bev_encoder` | BEVEncoder（可选） | SECOND + FPN 精炼 |
| `pose_embed` | nn.Parameter | 可学习 BEV 位置编码 |
| `deformable_transformer` | deformable_transformer_layer × N | 可变形注意力（含 DropPath） |
| `head_drop` | nn.Dropout | 预测头前的 Dropout |
| `translation_pred` | nn.Linear(embed_dim, 3) | 平移预测（rotation_only 时禁用） |
| `rotation_pred` | nn.Linear(embed_dim, 4) | 四元数旋转预测 |
| `loss_fn` | realworld_loss | 多项损失函数 |

**前向传播**：

```python
def forward(self, img, pc, gt_T_to_camera, init_T_to_camera,
            post_cam2ego_T, cam_intrinsic, masks=None, out_init_loss=False):
```

1. 图像分支：`cam2ego_T = inv(init_T)` → `img_branch(cam2ego_T, intrinsics, imgs)` → `cam_bev_feats`
2. 点云分支：`pc_branch(pc)` → `pc_bev_feats`
3. 特征融合：`conv_fuser(cam_bev, pc_bev)` → `+ pose_embed`
4. Transformer：`deformable_transformer(x)` → `mean(dim=1)` 全局池化
5. 预测头：`head_drop(x)` → `translation_pred(x)` + `rotation_pred(x)` → `get_T_matrix()`
6. 损失计算：`loss_fn(pred_translation, pred_rotation, pcs, gt_T, init_T, masks)`

返回 `(pred_T, init_loss, loss_dict)`。

### 2.2 图像分支：`img_branch/img_branch.py`

#### Cam2BEV 类

将图像投影到 BEV 空间，核心流程：

1. **SwinT 编码**：`SwinT_tiny_Encoder`（预训练 Swin-Tiny + FPN）提取多尺度特征
2. **LSS 深度估计**：预测每个像素的深度分布，构建 3D 视锥体
3. **BEV Pool**：CUDA 加速的体素聚合，将视锥体特征散射到 BEV 网格
4. **ProjectionHead**：`Linear → GELU → Linear → LayerNorm` 映射到统一特征空间

输出：`(B, 128, H_bev, W_bev)`

#### LSS 类

- **Lift**：深度网络预测 D 个深度 bin 的 softmax 分布，与特征相乘得到深度加权特征
- **Splat**：通过 `cam2ego_T` 和内参将视锥体点变换到 ego 坐标系，分配到 BEV 网格

### 2.3 点云分支：`pc_branch/pc_branch.py`

#### Lidar2BEV 类

1. **PointToVoxel**：spconv 体素化，`vsize_xyz` 由 `bev_settings` 计算
2. **SparseEncoder**：4 层 3D 稀疏卷积（16→32→64→128），使用 SparseBasicBlock 残差块
3. **Z 维度压缩**：`dense()` → reshape → `(B, C*Z, X, Y)`
4. **ProjectionHead**：映射到 128 维

输出：`(B, 128, H_bev, W_bev)`

### 2.4 损失函数：`losses/losses.py`

#### realworld_loss 类

```python
class realworld_loss(nn.Module):
    def __init__(self,
                 weight_translation=1.0,
                 weight_quat_norm=0.5,
                 weight_rotation=0.5,
                 weight_PCreproj=0.5,
                 enable_axis_loss=False,
                 weight_axis_rotation=0.3,
                 axis_weights=(3.0, 1.5, 1.0),
                 tail_topk_ratio=0.1,
                 tail_weight=0.5):
```

**损失项**：

| 损失 | 类 | 公式 | 权重 |
|------|----|------|------|
| 平移损失 | `translation_loss` | SmoothL1(Δt, gt_t) | 1.0 |
| 旋转损失 | `rotation_loss` | 四元数测地距离 | 0.5 |
| 四元数归一化 | `quat_norm_loss` | (‖q‖² − 1)² | 0.5 |
| 重投影损失 | `PC_reproj_loss` | ‖T⁻¹·T̂·p − p‖ | 0.5 |
| 轴旋转损失 | `AdaptiveAxisRotationLoss` | 逐轴 Euler + tail penalty | 0.3 |

#### AdaptiveAxisRotationLoss 类

```python
class AdaptiveAxisRotationLoss(nn.Module):
    def __init__(self, axis_weights=(3.0, 1.5, 1.0),
                 tail_topk_ratio=0.1, tail_weight=0.5):
```

- 将旋转误差分解为 Roll/Pitch/Yaw（ZYX Euler）
- 加权：Roll×3.0, Pitch×1.5, Yaw×1.0
- Tail penalty：对 batch 中最差 10% 样本额外施加 2× 惩罚
- 返回 `(loss, per_axis_dict)` 包含 `axis_roll_loss`, `axis_pitch_loss`, `axis_yaw_loss`

### 2.6 扰动生成：`tools.py`

```python
def generate_single_perturbation_from_T(
    T, angle_range_deg=20, trans_range=1.5,
    rotation_only=False,
    distribution='uniform',
    per_axis_prob=0.0,
    curriculum_scale=1.0,
    per_axis_weights=None):
```

**分布模式**：
- `uniform`：均匀分布采样角度和平移
- `truncated_normal`：截断正态分布（sigma_scale=0.5），集中在小扰动区域

**Per-axis 模式**：当 `per_axis_prob > 0` 时，以一定概率对单轴施加扰动，权重由 `per_axis_weights` 控制（默认 Roll 偏重）。

## 3. BEV 空间配置

`bev_settings.py` 通过环境变量支持 KITTI 和自定义数据集的不同配置：

| 参数 | KITTI | Custom | 环境变量 |
|------|-------|--------|----------|
| xbound | (-90, 90, 2.0) | (0, 200, 2.0) | `BEV_XBOUND_MIN/MAX` |
| ybound | (-90, 90, 2.0) | (-100, 100, 2.0) | `BEV_YBOUND_MIN/MAX` |
| zbound | (-10, 10, 20.0) | (-10, 10, step) | `BEV_ZBOUND_STEP` |
| d_conf | (1.0, 90.0, 1.0) | (1.0, 100.0, 1.0) | — |
| DATASET_TYPE | kitti | custom | `BEV_DATASET_TYPE` |

## 4. 训练脚本链

```
configs/*.yaml → batch_train.sh → start_training.sh → train_universal.sh → train_kitti.py
```

| 脚本 | 作用 |
|------|------|
| `configs/*.yaml` | 实验定义：defaults + experiments 覆盖 |
| `batch_train.sh` | 解析 YAML，循环实验，节点 rank 门控 |
| `start_training.sh` | 数据集路径解析，DDP vs 经典模式切换 |
| `train_universal.sh` | 构建参数，编译 bev_pool，启动 torchrun |
| `train_kitti.py` | DDP 初始化，AMP，训练循环，日志，检查点 |
