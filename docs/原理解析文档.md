# BEVCalib 原理解析文档

## 1. 概述

BEVCalib 是一个基于几何引导的鸟瞰图（BEV）表示的 LiDAR-Camera 外参标定方法。通过将图像和点云统一投影到 BEV 空间，利用几何一致性学习精确的 LiDAR→Camera 变换矩阵。

### 1.1 核心思想

1. **统一表示空间**：图像和点云分别投影到 BEV 空间，在同一坐标系下对齐
2. **几何引导**：利用 BEV 空间的几何结构引导特征学习和标定参数预测
3. **端到端学习**：直接从原始数据学习标定参数，无需手动特征提取

### 1.2 问题定义

给定：
- 图像 $I \in \mathbb{R}^{H \times W \times 3}$
- 点云 $P \in \mathbb{R}^{N \times 3}$
- 初始标定矩阵 $T_{init} \in SE(3)$（含噪声）

目标：预测精确的标定矩阵 $T_{gt} \in SE(3)$

## 2. 方法架构

### 2.1 整体流程

```
输入: 图像 + 点云 + 初始标定矩阵 T_init
  ↓
1. 图像分支 (Cam2BEV): SwinT → LSS → BEV Pool → (B, 128, H, W)
  ↓
2. 点云分支 (Lidar2BEV): Voxelize → SparseEncoder → (B, 128, H, W)
  ↓
3. 特征融合 (ConvFuser): concat → 1×1 Conv → (B, embed_dim, H, W)
  ↓
4. Transformer: Deformable Attention → 全局池化 → (B, embed_dim)
  ↓
5. 预测头: ΔT(3) + Δq(4) → 4×4 变换矩阵
  ↓
输出: 精确的标定矩阵 T_pred
```

### 2.2 图像到 BEV 的投影（Lift-Splat-Shoot）

#### Lift 阶段

1. **图像编码**：Swin Transformer Tiny + FPN 提取多尺度特征
2. **深度预测**：深度网络输出 D 个深度 bin 的 softmax 分布
   - 深度范围：KITTI $[1.0, 90.0]$m，Custom $[1.0, 100.0]$m，步长 $1.0$m
3. **视锥体构建**：为每个像素在每个深度 bin 创建 3D 点，通过内参和 $T_{init}^{-1}$ 变换到 ego 坐标系

#### Splat 阶段

1. **体素化**：将 3D 视锥体点分配到 BEV 网格
2. **BEV Pool**：CUDA 加速的特征聚合，将深度加权特征散射到 BEV 网格
3. **Z 维度拼接**：多层 Z 特征拼接为单层 BEV 特征

#### 投影头

`Linear → GELU → Linear → LayerNorm`，输出 $(B, 128, H_{bev}, W_{bev})$

### 2.3 点云到 BEV 的投影

1. **体素化**：spconv `PointToVoxel`，体素大小由 `bev_settings` 配置
2. **稀疏卷积**：4 层 SparseEncoder（16→32→64→128），使用 SparseBasicBlock 残差块
3. **Z 压缩**：`dense()` 后 reshape，将 Z 维度合并到通道维度
4. **投影头**：同图像分支，输出 $(B, 128, H_{bev}, W_{bev})$

### 2.4 特征融合

- **ConvFuser**：`concat(img_bev, pc_bev)` → 1×1 Conv → BN → ReLU
- **BEVEncoder**（可选）：SECOND + SECONDFPN 进一步精炼
- **位置编码**：可学习的 `pose_embed` 参数

### 2.5 Transformer 编码

#### 可变形 Transformer（默认）

- `deformable_transformer_layer` × `num_layers`
- 每层：BatchNorm → DeformableAttention → DropPath → BatchNorm → MLP → DropPath
- DropPath rate 可配置（默认 0.1）
- 全局 mean pooling → $(B, embed\_dim)$

#### 标准 Transformer（可选）

- `nn.TransformerEncoder`，层数 = `num_layers × 4`
- 支持 BEV 掩码处理无效区域

### 2.6 标定参数预测

- **Head Dropout**：预测前施加 Dropout（默认 0.1）
- **旋转预测**：`Linear(embed_dim, 4)` → 四元数，归一化到单位四元数
- **平移预测**：`Linear(embed_dim, 3)`（`rotation_only` 模式下禁用）
- **矩阵构建**：$T_{pred} = T_{translation} \cdot T_{rotation}$
- **最终输出**：$T_{gt\_expected} = T_{pred}^{-1} \cdot T_{init}$

## 3. 损失函数

### 3.1 总损失

$$\mathcal{L}_{total} = w_t \mathcal{L}_{trans} + w_r \mathcal{L}_{rot} + w_q \mathcal{L}_{quat} + w_p \mathcal{L}_{reproj} + w_a \mathcal{L}_{axis} + w_{tail} \mathcal{L}_{tail}$$

默认权重：$w_t=1.0$, $w_r=0.5$, $w_q=0.5$, $w_p=0.5$, $w_a=0.3$, $w_{tail}=0.5$

### 3.2 基础损失

| 损失 | 公式 |
|------|------|
| 平移损失 | $\mathcal{L}_{trans} = \text{SmoothL1}(t_{pred}, t_{gt})$ |
| 旋转损失 | $\mathcal{L}_{rot} = \min(\|q_1 - q_2\|, \|q_1 + q_2\|)$ |
| 四元数归一化 | $\mathcal{L}_{quat} = (\|q_{pred}\|^2 - 1)^2$ |
| 重投影损失 | $\mathcal{L}_{reproj} = \frac{1}{N}\sum_j \|T_{gt}^{-1} T_{pred}^{-1} T_{init} p_j - p_j\|_2$ |

### 3.3 AdaptiveAxisRotationLoss（V11 新增）

将旋转误差分解为 Roll/Pitch/Yaw 三个轴，分别施加加权损失：

$$\mathcal{L}_{axis} = w_{roll} \cdot |\Delta\theta_{roll}| + w_{pitch} \cdot |\Delta\theta_{pitch}| + w_{yaw} \cdot |\Delta\theta_{yaw}|$$

默认权重：Roll=3.0, Pitch=1.5, Yaw=1.0（Roll 通常最难学习）

**Tail Penalty**：对 batch 中误差最大的 top-k 样本（默认 10%）施加额外惩罚：

$$\mathcal{L}_{tail} = w_{tail} \cdot \text{mean}(\text{topk}(\mathcal{L}_{axis}, k=\lfloor 0.1 \cdot B \rfloor))$$

## 4. 训练策略

### 4.1 扰动生成

对 GT 标定矩阵添加随机扰动模拟初始标定不准确性：

| 参数 | 说明 | 典型值 |
|------|------|--------|
| `distribution` | 采样分布 | `uniform` / `truncated_normal` |
| `angle_range_deg` | 旋转扰动范围 | 5°~10° |
| `trans_range` | 平移扰动范围 | 0.15~0.3m |
| `per_axis_prob` | 单轴扰动概率 | 0.3 |
| `per_axis_weights` | 轴采样权重 [R,P,Y] | [3.0, 1.5, 1.0] |

- **uniform**：均匀分布采样
- **truncated_normal**：截断正态分布（sigma_scale=0.5），集中在小扰动区域

### 4.2 数据增强

| 增强方式 | 参数 | 说明 |
|----------|------|------|
| 点云抖动 | `augment_pc_jitter` σ=0.01 | 高斯噪声 |
| 点云随机丢弃 | `augment_pc_dropout` p=0.05 | 随机丢弃点 |
| 颜色抖动 | `augment_color_jitter` | 亮度/对比度/饱和度 |

### 4.3 优化器与调度器

- **优化器**：AdamW，`backbone_lr_scale`（默认 0.1）对 backbone 使用更小学习率
- **调度器**：
  - **Warmup + Cosine**（推荐）：`LinearLR` warmup（3~5 epochs）→ `CosineAnnealingWarmRestarts`
  - **StepLR**（可选）：每 N epochs 衰减 0.5×

### 4.4 正则化

| 方法 | 参数 | 说明 |
|------|------|------|
| DropPath | `drop_path_rate` 0.1~0.2 | Transformer 层随机深度 |
| Head Dropout | `head_dropout` 0.1~0.3 | 预测头前 Dropout |
| Early Stopping | `early_stopping_patience` 15~20 | 验证集无改善时停止 |

### 4.5 分布式训练

- **DDP**（DistributedDataParallel）：多 GPU / 多节点
- **AMP**（Automatic Mixed Precision）：FP16 加速
- **GradScaler**：混合精度梯度缩放

## 5. BEV 空间配置

| 参数 | KITTI | Custom |
|------|-------|--------|
| X 范围 | [-90, 90]m, 2.0m/cell | [0, 200]m, 2.0m/cell |
| Y 范围 | [-90, 90]m, 2.0m/cell | [-100, 100]m, 2.0m/cell |
| Z 范围 | [-10, 10]m, 20.0m/cell | [-10, 10]m, step 可配 |
| BEV 网格 | 90×90 | 100×100 |
| 深度范围 | [1, 90]m | [1, 100]m |

Z 维度步长通过 `BEV_ZBOUND_STEP` 环境变量控制，影响 Z 方向的体素分辨率和模型容量。

## 6. 评估指标

| 指标 | 说明 | 目标 |
|------|------|------|
| 旋转误差 | Roll/Pitch/Yaw 欧拉角误差（度） | < 0.5° |
| 平移误差 | XYZ 方向 L2 距离（米） | < 0.05m |
| 像素位移 | 投影点的平均像素偏移 | < 1px (Excellent) |
| 改善百分比 | (init_error - pred_error) / init_error | > 90% |
